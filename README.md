# Course Projects Portfolio

This repository contains selected academic and technical projects in data analytics, machine learning, database systems, cybersecurity, optimization, stream processing, and cloud data warehouse design.

## Projects Overview

| Project | Tools | Summary |
|---|---|---|
| Linear Programming Solver | Python, NumPy, Linux | Implemented a simplex-based LP solver to classify problems as optimal, infeasible, or unbounded and output optimal solutions when feasible. |
| Handwritten Digit Recognition using Deep Networks | Python, PyTorch, OpenCV | Built and evaluated CNN models for handwritten digit recognition, including model training, testing, filter visualization, custom image prediction, and live video prediction. |
| Music Auto Generation | Python, Keras, TensorFlow | Designed an LSTM-based model to learn sequential note and chord patterns from MIDI files and generate new piano-style music. |
| Mining Stack Overflow Data | Scala, Spark | Applied FP-Growth and LDA topic modeling to analyze programming language co-occurrence patterns from Stack Overflow data. |
| Comparisons Between Large Collections of Sets | Scala, Spark | Implemented shingling, MinHash, Jaccard similarity, and locality-sensitive hashing to identify similar record pairs efficiently. |
| Processing Streams | Scala | Implemented Bloom Filters, Flajolet-Martin distinct counting, and Reservoir Sampling for memory-efficient stream processing. |
| Network Activity Analysis on E-Commerce Platform | PostgreSQL | Analyzed web access logs to detect SQL injection, brute-force login attempts, URL enumeration, and suspicious request patterns. |
| Data Warehouse – RCMP PTSD Longitudinal Study | SQL, BigQuery, Tableau, Matillion | Designed a cloud-based data warehouse and ETL architecture for multi-source longitudinal research data. |
| Movie Exploration | MySQL, Neo4j | Designed a relational and graph database model to explore movie recommendations based on customer rental history, country, category, and actor preferences. |
| Dimensionality Reduction | MATLAB | Applied PCA and Fisher Linear Discriminant to reduce high-dimensional data and evaluated results using Bayes classification and neural networks. |
| Correlation and Mutual Information | MATLAB | Conducted Gaussian hypothesis testing, Bayes classification, neural network evaluation, and feature correlation analysis. |
| Analysis on Time Series Data | MATLAB | Applied stationarity testing, KS-test, median filtering, and wavelet-based denoising techniques for time series data. |
| Dealing with Imbalanced Dataset with Regression Models | MATLAB | Built weighted logistic regression models using Gradient Descent and Newton’s method for imbalanced classification tasks. |

## Main Skills Demonstrated

- Python, SQL, Scala, MATLAB
- PyTorch, Keras, TensorFlow
- MySQL, PostgreSQL, Neo4j
- Apache Spark, Spark MLlib, Spark ML
- Power BI / Tableau-style reporting concepts
- Machine learning and deep learning
- Data warehouse and ETL architecture
- Cybersecurity log analysis
- Optimization algorithms
- Stream processing and approximate algorithms
