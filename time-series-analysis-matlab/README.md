# Analysis on Time Series Data | MATLAB

This project explores time series data analysis and denoising techniques using MATLAB. The work focuses on testing statistical properties of time series data, detecting spurious samples, and applying filtering methods to reduce noise.

## Project Overview

The main goals of this project were to:

- Check whether time series data is stationary or ergodic using hypothesis testing
- Apply the Kolmogorov-Smirnov test to compare data distributions
- Detect and remove spurious data samples using median filtering
- Experiment with MATLAB Wavelet Toolbox for denoising

## Tools Used

- MATLAB
- MATLAB Live Script
- Wavelet Toolbox
- Statistical hypothesis testing
- Median filtering

## Key Methods

### 1. Stationarity and Ergodicity Testing

The project used statistical tests to examine whether the time series data maintained consistent distributional properties over time.

### 2. KS-Test

The Kolmogorov-Smirnov test was used to compare sample distributions and support hypothesis testing on the time series data.

### 3. Median Filtering

Median filtering was implemented to identify and reduce the effect of abnormal or spurious data points.

### 4. Wavelet Denoising

The MATLAB Wavelet Toolbox was used to experiment with signal denoising techniques and compare their effects on noisy time series data.
